# Tecla — Sitio web

Landing de **Tecla**, con un switch entre las dos unidades de negocio:

- **Tecla Soporte** — servicio técnico informático (reparaciones, sistemas, backup, armado).
- **Tecla para docentes** — ayuda a medida para docentes del sistema educativo bonaerense.

Es un sitio estático (un solo `index.html` autocontenido). Etapa de validación: solo recolecta contactos, sin planes ni precios.

## Estructura

```
index.html            → el sitio completo (HTML + CSS + JS en un archivo)
logo/                 → variantes del logo en SVG
fotos/                → fotos del equipo (gaston.jpg, andres.jpg)
design-system.md      → colores, tipografías y guía de marca
SETUP-formulario.md   → cómo activar la captación de contactos (Formspree)
```

## Verlo localmente

Abrí `index.html` en cualquier navegador. No necesita servidor.

## Publicado en

GitHub Pages: _(completar con la URL cuando esté activo, ej. https://gaston-perez-art.github.io/tecla/)_

## Formulario de contacto

Por defecto está en **modo demo** (no guarda nada). Para recibir los contactos, seguí `SETUP-formulario.md` (Formspree, gratis).
